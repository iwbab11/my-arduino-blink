# Arduino Blink & SOS Project
 Учебный проект для освоения Git в разработке под Arduino.
 ## Оборудование
 * Arduino Uno
 * 2 светодиода (пины 9 и 12)
 * Резисторы 220 Ом
 * Макетная плата
 ## Функциональность
 1. Основной режим (main): Одновременное мигание светодиодов.
 2. Режим SOS (ветка feature-sos): Сигнал SOS одним из светодиодов.
